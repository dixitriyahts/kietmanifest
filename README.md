# KIET School Of Management Manifest Website

Welcome to the official repository for the KIET School Of Management Manifest website. This repository contains the codebase for the Manifest 2024 event.

## Description

This website serves as the digital platform for the Manifest 2024 event organized by the KIET School Of Management. The website provides information about the event details, objectives, activities, contacts, and more.

## Features

- **Responsive Design**: The website is designed to be responsive and works seamlessly on various devices.
- **Navigation**: Easy navigation with a menu that directs users to different sections of the website.
- **Event Details**: Detailed information about the Manifest 2024 event, including objectives, activities, and document links.
- **Contact Information**: Contact details of the organizing team and coordinators are provided for easy communication.
- **Interactive Elements**: Collapsible sections for objectives and clickable images for glimpses of KIET.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/dixitriyahts/kietmanifest.git
   ```

2. Navigate to the project directory:

   ```bash
   cd kiet-manifest
   ```

3. Open the `index.html` file in your web browser to view the website locally.

## Usage

1. Browse through the website sections using the navigation menu.
2. Click on the links/buttons to explore details about the Manifest 2024 event.
3. Contact the organizing team or coordinators using the provided contact information.
4. Explore the document links and glimpses of KIET for additional information.
